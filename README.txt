Задача:
Написать программу, которая из имеющегося массива строк формирует массив из строк, длина которых меньше либо равна 3 символам.

Блок-схема алгоритма:
1. Создаем исходный массив temp из всех возможных символов: цифры, заглавные буквы, прописные буквы, символы;
2. Вводим размерность нового массива - size;
3. Вводим строчный массив count - для вывода нового массива, длина элементов которого меньше или равна 3 символам;
4. Вводим переменную i - для счетчика каждого элемента нового массива, чтобы он не выходил за размерность массива - size;
4.1. Вводим переменную random (генерируется автоматически) - для указания размерности каждого i-го элемента массива (максимальное количество символов в i-м элементе массива указал как 8);
4.2. Вводим переменную j - для счетчика размерности i-го элемента массива (не больше random);
4.2.1. Вводим переменную pos (генерируется автоматически) - для создания каждого j-го значения i-го элемента массива, pos - указывает позиция символа в исходном массиве temp;
4.2.2. Соединяем pos-ый элемент массива temp к каждому j-му элементу i-го элемента массива;
4.2.3. Когда значение j приравнивается значению random (т.е. когда количество символов соответствует размерности random от i-го элемента массива) - выходим из j-го цикла;
4.3. Делаем проверку на длину i-го элемента массива: длина i-го элемента массива меньше 4? 
4.3.1. Если меньше - то соединяем i-й элемент в строчный массив count;
4.4. Когда значение i приравнивается значению size (т.е. когда количество элементов массива соответствует размерности size массива) - выходим из i-го цикла;
5. Выводим массив.
